# web4 - Лабораторная работа 4

Комплексное веб-приложение с интеграцией всех предыдущих технологий и новых подходов.

## 📋 Описание

Полнофункциональное веб-приложение на базе современного Java технологического стека с фронтенд-компонентом и микросервисной архитектурой.

## 🛠️ Технологический стек

- **Java**
- **Maven** - управление проектом и зависимостями
- **Spring Framework**
- **Docker Compose** - контейнеризация и оркестрация
- **Frontend** - REACT.js + Redux Toolkit
- **Микросервисная архитектура**

## 🚀 Запуск

### С использованием Docker Compose

```bash
# Сборка и запуск всех сервисов
docker-compose -f compose.yaml up -d

# Остановка сервисов
docker-compose -f compose.yaml down
```

### Локальная сборка

```bash
# Сборка проекта
./mvnw clean install

# Запуск приложения
./mvnw spring-boot:run
```

## 📝 Функциональность

- Комплексная система определения попадания точки в область
- Интеграция фронтенд и бэкенд компонентов
- REST API
- Микросервисная архитектура
- Контейнеризация с Docker
